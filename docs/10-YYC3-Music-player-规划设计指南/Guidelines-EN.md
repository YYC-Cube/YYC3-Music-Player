# Guidelines for AI Music Creation Full Closed-Loop Workflow

> **Important Note**: D-Music is a bilingual (Chinese-English) system, with **Chinese** set as the default display language. You can switch the interface language at any time in the settings to suit your preference.

---

## 1. Pre-Creation: Big Data Positioning & Inspiration Anchoring

### 1.1 Data Insight (Key Operations)
- Navigate to **My** → Review 30-day work data: likes, collections, plays, comment keywords, and audience demographics.
- Identify high-performing style combinations, theme preferences, and audience age/geographic distribution.

### 1.2 Inspiration Anchoring (Prompt Template)
```markdown
【Big Data Creation Anchoring】:
- Core Insight: 30-day data shows the "Jiangnan Elegance + Wandering Folk" style combination has 2.3x more likes than other styles, with a primary audience of 18-25-year-old females who prefer "urban nostalgia" and "healing longing" themes.
- Creative Direction: Next work theme: *Urban Nostalgia*; Style combination: *Jiangnan Elegance + Wandering Folk*; Vocal tone: *Cool female voice*.
- Monetization Expectation: Prioritize for copyright distribution and digital record production.
```

---

## 2. During Creation: From Inspiration to Production

### 2.1 Minimalist Songwriting (Rapid Generation)
- **Key Operations**: Enter **AI Creation** → Select **Minimalist Songwriting** → Input prompt → Click generate → Auto-save to **My > Draft Box**.
- **Prompt Template**:
```markdown
A Wandering Folk + Jiangnan Elegance song about urban nostalgia, in Chinese, with guitar and guzheng, with a gentle and healing mood. Auto-save to draft box after generation.
```

### 2.2 Master Songwriting (Deep Customization)
- **Key Operations**: Enter **AI Creation** → Select **Master Songwriting** → Fill in song name/lyrics/ideas → Select style combination, vocal tone, audio reference → Generate → Auto-save to **My**.
- **Prompt Template**:
```markdown
【Song Name】: "Wind at the Alley Entrance"
【Lyrics/Creative Idea】: I want to write a song about urban nostalgia, telling the story of young people working in big cities missing the old streets of their hometown. Use guitar and guzheng to express a gentle and healing mood, with string arrangements in the chorus. Auto-save to draft box after generation.
【Music Style Combination】:
- Primary Style: Wandering Folk
- Secondary Style: Jiangnan Elegance
- Style Dimension: Mood + Genre
- Creation Mode: Precise Specification
【Vocal Tone】:
- Base Option: Female
- Detailed Description: Cool female voice
【Audio Reference】:
- Input Method: Upload Audio
- Reference Note: Upload a 60-second ancient-style flute solo
- Expected Impact: Retain the soothing tone of the reference melody as the foundation for the song's verse.
```

### 2.3 MV Creation (Audio-Visual Integration)
- **Key Operations**: Enter **AI Creation** → Select **AI MV** → Select target song → Input prompt → Generate → Auto-save to **My > MV Category**.
- **Prompt Template**:
```markdown
【MV Type】: AI MV
【Target Work】: "Wind at the Alley Entrance"
【Creative Requirements】: Urban nostalgic style, visual elements include old streets, street lamps, fallen leaves, and homebound pedestrians, with warm yellow and slate gray tones, creating a gentle and healing atmosphere. Auto-save to public works (MV category) after generation.
```

---

## 3. Post-Creation: Work Management & Iteration

### 3.1 Work Management (Categorization & Organization)
- **Key Operations**: Enter **My** → Select **Works** → Filter by type/time → Set privacy/move to album.
- **Prompt Template**:
```markdown
【Work Management Operations】:
- Filter Criteria: Type: Music; Status: Draft; Time: Last 7 days
- Privacy Settings: Set "Wind at the Alley Entrance" draft to private, optimize before publishing
- Batch Operations: Move 3 folk works to the "Urban Nostalgia" album
```

### 3.2 Secondary Creation (Iterative Optimization)
- **Key Operations**: Enter **My** → Select target work → Click **Recreate** / **AI Cover** / **Produce MV** / **Extract Accompaniment**.
- **Prompt Template**:
```markdown
【Single Work Secondary Creation】:
- Target Work: "Wind at the Alley Entrance"
- Operation Type: Recreate
- Creative Requirements: Based on the current version, replace the guzheng in the arrangement with an electronic synthesizer, adjust the style to "Jiangnan Elegance + Modern Electronic", and regenerate the chorus. Auto-save as a new version after generation.
```

---

## 4. Copyright & Monetization: From Creation to Value

### 4.1 Originality Certification (Copyright Protection)
- **Key Operations**: Enter **My** → Select target work → Click **Originality Certification** → Submit application → Wait for approval to obtain certificate.
- **Prompt Template**:
```markdown
【Originality Certification Application】:
- Target Work: "Wind at the Alley Entrance"
- Originality Statement: This work was created with AI assistance and manually optimized. I hold full copyright to this work and have not infringed on any third-party rights.
- Intended Use: Commercial distribution, short video soundtracks, live background music, etc.
```

### 4.2 Music Distribution (Commercial Monetization)
- **Key Operations**: Enter **My** → Select target work → Click **Music Distribution Service** → Fill in distribution info → Select platforms → Submit for review.
- **Prompt Template**:
```markdown
【Music Distribution Application】:
- Target Work: "Wind at the Alley Entrance"
- Distribution Info:
  - Song Name: "Wind at the Alley Entrance"
  - Cover: Illustration of an old urban street
  - Description: A healing folk song about urban nostalgia, woven with guitar and guzheng to evoke the tenderness of homecoming.
  - Copyright Ownership: Personal ownership, national copyright certificate applied for
- Distribution Platforms: NetEase Cloud Music, QQ Music, Kugou Music
- Sales Model: Free streaming (revenue share based on plays)
```

### 4.3 Digital Records (Collectible Value)
- **Key Operations**: Enter **My** → Select target work → Click **Digital Records** → Design cover/additional benefits → Set limited quantity → Publish for sale.
- **Prompt Template**:
```markdown
【Digital Record Release】:
- Target Work: "Wind at the Alley Entrance"
- Design Plan:
  - Cover: Dynamic illustration featuring old streets, street lamps, and homebound pedestrians
  - Additional Benefits: Exclusive signature, behind-the-scenes creation story, accompaniment file

```

---

## 5. Data Review: Iterating for Next Creation

### 5.1 Data Collection (Key Operations)
- Navigate to **My** → Review work data: likes, collections, plays, comment keywords, and audience demographics.
- Record core metrics: performance of style combinations, themes, vocal tones, and monetization methods.

### 5.2 Creative Iteration (Prompt Template)
```markdown
【Big Data Creation Iteration】:
- Data Review: After release, "Wind at the Alley Entrance" has exceeded 100,000 plays, with comment keywords focusing on "healing", "nostalgia", and "Jiangnan", and a primary audience of 18-25-year-old females.
- Creative Adjustments:
  - Next work theme: *Urban Healing*, continuing the "Jiangnan Elegance + Wandering Folk" style combination.
  - Vocal Tone: Cool female voice.
- Monetization Optimization: Prioritize works in this style combination for digital record production to enhance collectible value.
```

---

## 6. Full Closed-Loop Workflow
1. **Data Insight** → 2. **Inspiration Anchoring** → 3. **Work Generation** → 4. **Work Management** → 5. **Secondary Creation** → 6. **Copyright Protection** → 7. **Commercial Monetization** → 8. **Data Review** → Return to Step 1 to form a closed loop.
